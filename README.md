## 🚀 Cara Pakai

1. Buka URL Netlify project ini di browser (Chrome / Brave)
2. Pastikan ekstensi **Phantom Wallet** sudah terinstall → [phantom.app](https://phantom.app)
3. Klik **CONNECT PHANTOM** → approve di popup
4. Ketik token di kolom Search → klik **SCAN**
5. Atur konfigurasi (TP, SL, Strategy, Chain)
6. Klik **▶ START AUTO-TRADE** untuk mulai bot otomatis
